Crystals Collector Game!

Welcome to the Crystals Collector Game!

You will be given a random number at the beginning of the game. The object of the game is to try to pick the four crystals that add up to the number given.

Each crystal will be given a random number and will be revealed to you after you select it.

You will lose if you go over the number given and the game will restart and give the crystals different values.

Good Luck!
